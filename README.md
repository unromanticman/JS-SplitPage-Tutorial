# JS-SplitPage-Tutorial
用javascript達成網頁分頁的教學
### Intro
只需要控制getSplitPage()函式內四個參數即可控制分頁的呈現
<pre>
//物品總數
var count = 36;
//當前頁面
var onpage = 7; 
//一個頁面顯示幾個物品
var file = 5;
//一個頁面顯示前後幾個頁面
var range = 5;
</pre>
